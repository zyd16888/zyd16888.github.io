☆自定义站点内容搜索

安装 hexo-generator-searchdb，在站点的根目录下执行以下命令：

npm install hexo-generator-searchdb --save
编辑 站点配置文件，新增以下内容到任意位置：

search:
path: search.xml
field: post
format: html
limit: 10000
编辑 主题配置文件，启用本地搜索功能：

# Local search
local_search:
enable: true

☆链接持久化

permalink_defaults:
abbrlink:
alg: crc32 # 算法：crc16(default) and crc32
rep: hex # 进制：dec(default) and hex
